## 一、数据库概述

### 1.1 数据库信息

| 项目 | 说明 |
|------|------|
| 数据库类型 | MySQL 8.0 |
| 字符集 | utf8mb4 |
| 排序规则 | utf8mb4_unicode_ci |
| 存储引擎 | InnoDB |
| 连接池 | Druid |

### 1.2 设计原则

1. **服务拆分**：每个微服务独立数据库，实现数据隔离
2. **逻辑删除**：使用delete_flag字段实现软删除
3. **审计字段**：统一使用create_time、update_time字段
4. **索引优化**：针对高频查询字段建立索引
5. **字段冗余**：适当冗余减少关联查询
