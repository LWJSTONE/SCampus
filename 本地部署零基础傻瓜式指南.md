## 二、安装必要软件

### 2.1 安装 JDK 1.8

#### Windows系统

1. **下载JDK**
   - 访问Oracle官网或采用开源版本：
   - 推荐下载：[Adoptium Temurin JDK 8](https://adoptium.net/temurin/releases/?version=8)
   - 选择Windows x64版本，下载.msi安装包

2. **安装JDK**
   ```
   双击下载的.msi文件
   → 点击"Next"
   → 选择安装路径（建议默认）
   → 点击"Install"
   → 等待安装完成
   ```

3. **配置环境变量**（重要！）

   **步骤1：设置JAVA_HOME**
   ```
   右键"此电脑" → 属性 → 高级系统设置 → 环境变量
   → 在"系统变量"中点击"新建"
   → 变量名：JAVA_HOME
   → 变量值：C:\Program Files\Java\jdk1.8.0_xxx（你的JDK安装路径）
   → 点击"确定"
   ```

   **步骤2：配置Path**
   ```
   在"系统变量"中找到"Path"变量
   → 点击"编辑"
   → 点击"新建"
   → 输入：%JAVA_HOME%\bin
   → 点击"确定"
   ```

4. **验证安装**
   ```
   打开命令提示符（Win+R，输入cmd）
   输入：java -version
   
   显示以下内容表示成功：
   java version "1.8.0_xxx"
   Java(TM) SE Runtime Environment...
   ```

### 2.2 安装 Maven 3.6+

1. **下载Maven**
   - 访问：https://maven.apache.org/download.cgi
   - 下载：apache-maven-3.9.6-bin.zip

2. **解压安装**
   ```
   解压到：D:\apache-maven-3.9.6（或你喜欢的路径）
   ```

3. **配置环境变量**
   ```
   新建系统变量：
   变量名：MAVEN_HOME
   变量值：D:\apache-maven-3.9.6
   
   编辑Path变量，添加：
   %MAVEN_HOME%\bin
   ```

4. **验证安装**
   ```
   cmd输入：mvn -version
   
   显示以下内容表示成功：
   Apache Maven 3.9.6...
   ```

### 2.3 安装 MySQL 8.0

1. **下载MySQL**
   - 访问：https://dev.mysql.com/downloads/mysql/
   - 选择Windows版本，下载MSI安装包

2. **安装MySQL**
   ```
   双击.msi文件
   → 选择"Developer Default"或"Server only"
   → 点击"Execute"执行安装
   → 配置MySQL Server：
     - 设置root密码（记住这个密码！）
     - 端口保持默认3306
   → 完成安装
   ```

3. **验证安装**
   ```
   打开MySQL Command Line Client
   输入root密码
   
   显示以下内容表示成功：
   mysql>
   ```

### 2.4 安装 Redis（Windows版）

1. **下载Redis**
   - 访问：https://github.com/tporadowski/redis/releases
   - 下载：Redis-x64-5.0.14.1.msi

2. **安装Redis**
   ```
   双击.msi文件
   → 选择安装路径
   → 勾选"Add Redis to PATH"
   → 点击Install
   ```

3. **启动Redis服务**
   ```
   方法1：通过服务管理器
   Win+R → services.msc → 找到Redis → 右键启动
   
   方法2：通过命令行
   cmd输入：redis-server
   
   验证：
   新开一个cmd窗口，输入：redis-cli ping
   返回PONG表示成功
   ```

### 2.5 安装 Nacos 2.3.x

1. **下载Nacos**
   - 访问：https://github.com/alibaba/nacos/releases
   - 下载：nacos-server-2.3.2.zip

2. **解压并启动**
   ```
   解压到：D:\nacos
   
   Windows单机模式启动：
   cd D:\nacos\bin
   startup.cmd -m standalone
   ```

3. **访问Nacos控制台**
   ```
   浏览器访问：http://localhost:8848/nacos
   
   默认账号：nacos
   默认密码：nacos
   ```

### 2.6 安装 Node.js 18+

1. **下载Node.js**
   - 访问：https://nodejs.org/
   - 下载LTS版本（18.x或20.x）

2. **安装Node.js**
   ```
   双击.msi文件
   → 勾选"Automatically install the necessary tools..."
   → 点击Install
   ```

3. **验证安装**
   ```
   cmd输入：
   node -v     # 显示v18.x.x
   npm -v      # 显示9.x.x
   ```

4. **配置npm镜像（可选，加速下载）**
   ```
   npm config set registry https://registry.npmmirror.com
   ```

## 四、初始化数据库

### 4.1 打开MySQL命令行

```
方式1：MySQL Command Line Client
开始菜单 → MySQL → MySQL 8.0 Command Line Client

方式2：命令行
cmd → mysql -u root -p → 输入密码
```

### 4.2 执行SQL脚本

**按顺序执行以下脚本：**

```sql
-- 1. 创建数据库
source D:/projects/SCampus/doc/databases.sql;

-- 2. 创建表结构
source D:/projects/SCampus/doc/schema.sql;

-- 3. 导入初始数据
source D:/projects/SCampus/doc/data.sql;
```

**或者在MySQL Workbench中：**
1. 打开MySQL Workbench
2. 连接到本地MySQL
3. File → Run SQL Script → 选择对应的.sql文件

### 4.3 验证数据库

```sql
-- 查看创建的数据库
SHOW DATABASES;

-- 应该看到以下数据库：
-- forum_auth_db
-- forum_user_db
-- forum_category_db
-- forum_post_db
-- forum_comment_db
-- forum_interaction_db
-- forum_report_db
-- forum_stats_db
-- forum_notify_db
-- forum_file_db

-- 查看用户表数据
USE forum_user_db;
SELECT * FROM sys_user;
```

## 五、启动后端服务

### 5.1 确认中间件已启动

在启动后端服务前，请确保以下服务已启动：

| 服务 | 端口 | 状态检查 |
|------|------|----------|
| MySQL | 3306 | `mysql -u root -p` 能登录 |
| Redis | 6379 | `redis-cli ping` 返回PONG |
| Nacos | 8848 | 访问 http://localhost:8848/nacos |

### 5.2 编译项目

```bash
# 进入项目根目录
cd D:\projects\SCampus

# 编译打包（跳过测试）

# 首次编译可能需要下载依赖，请耐心等待（约5-10分钟）
```

## 八、常见问题解答

### Q1: mvn package 报错 "Could not find artifact"

**原因**：Maven无法下载依赖

**解决方案**：
```bash
# 配置阿里云镜像
# 编辑 D:\apache-maven-3.9.6\conf\settings.xml

# 在<mirrors>标签内添加：
<mirror>
  <id>aliyun</id>
  <mirrorOf>*</mirrorOf>
  <name>Aliyun Maven</name>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

### Q2: 服务启动报错 "Communications link failure"

**原因**：无法连接MySQL

**解决方案**：
1. 检查MySQL服务是否启动
2. 检查数据库连接配置（application.yml）
3. 确认数据库用户名密码正确

### Q3: 服务启动报错 "Unable to connect to Redis"

**原因**：无法连接Redis

**解决方案**：
```bash
# 检查Redis是否启动
redis-cli ping

# 如果未启动，启动Redis
redis-server
```

### Q4: 服务注册不到Nacos

**原因**：Nacos未启动或地址配置错误

**解决方案**：
1. 确认Nacos已启动：访问 http://localhost:8848/nacos
2. 检查bootstrap.yml中的Nacos地址配置

### Q5: 前端npm install报错

**原因**：网络问题或Node版本问题

**解决方案**：
```bash
# 切换npm镜像
npm config set registry https://registry.npmmirror.com

# 清除缓存重试
npm cache clean --force
npm install
```

### Q6: 前端页面打开空白

**原因**：后端服务未启动或网关未配置

**解决方案**：
1. 确认所有后端服务已启动
2. 检查网关服务是否正常运行
3. 查看浏览器控制台错误信息

### Q7: 登录提示密码错误

**原因**：密码已修改或数据库未初始化

**解决方案**：
```sql
-- 重置管理员密码
USE forum_user_db;
UPDATE sys_user SET password = '$2a$10$N.zmdr9k7uOCQb376NoUnuTJ8iAt6Z5EHsM8lE9lBOsl7iAt6Z5EH' WHERE username = 'admin';
-- 密码将被重置为 admin123
```
