## 一、系统概述

### 1.1 系统简介

SCampus是一款面向高校师生的校园论坛系统，提供帖子发布、评论互动、消息通知等功能。系统支持三种角色：普通用户、版主、管理员，每种角色拥有不同的权限。

## 二、用户注册与登录

### 2.1 用户注册

1. 访问系统首页
2. 点击右上角【注册】按钮
3. 填写注册信息：
   - **用户名**：4-20位字母、数字或下划线
   - **密码**：6-20位，建议包含字母和数字
   - **确认密码**：再次输入密码
   - **邮箱**：用于找回密码
   - **验证码**：输入图片验证码
4. 点击【注册】按钮完成注册
5. 注册成功后自动跳转到登录页面

![注册页面示意](./images/register.png)

### 2.2 用户登录

1. 访问系统首页
2. 点击右上角【登录】按钮
3. 输入用户名和密码
4. 输入验证码
5. 点击【登录】按钮
6. 登录成功后跳转到首页

**默认管理员账号**：
- 用户名：admin
- 密码：admin123

![登录页面示意](./images/login.png)

### 2.3 找回密码

1. 在登录页面点击【忘记密码】
2. 输入注册时使用的邮箱
3. 获取并输入邮箱验证码
4. 设置新密码
5. 完成密码重置

## 三、普通用户功能

### 3.1 浏览帖子

#### 3.1.1 首页帖子列表

- 登录后自动显示首页帖子列表
- 帖子按时间倒序排列
- 置顶帖和精华帖优先显示
- 支持分页浏览

#### 3.1.2 按版块浏览

1. 点击左侧导航栏的版块名称
2. 显示该版块下的所有帖子
3. 可按时间、热度、评论数排序

#### 3.1.3 搜索帖子

1. 点击顶部搜索框
2. 输入关键词
3. 按回车或点击搜索按钮
4. 显示搜索结果列表

### 3.2 发布帖子

#### 3.2.1 发布步骤

1. 点击左侧【发布帖子】按钮
2. 选择目标版块
3. 填写帖子标题（必填，5-100字）
4. 编辑帖子内容（支持富文本）
5. 添加标签（可选，最多5个）
6. 上传图片（可选）
7. 选择是否匿名发布
8. 点击【发布】按钮

#### 3.2.2 帖子编辑器功能

| 功能 | 说明 |
|------|------|
| 标题格式 | 支持H1-H6标题 |
| 文本样式 | 加粗、斜体、下划线、删除线 |
| 列表 | 有序列表、无序列表 |
| 链接 | 插入超链接 |
| 图片 | 上传或粘贴图片 |
| 代码 | 代码块（支持语法高亮） |
| 表格 | 插入表格 |
| 表情 | 插入表情符号 |

### 3.3 查看帖子详情

1. 点击帖子标题进入详情页
2. 显示帖子完整内容
3. 显示作者信息
4. 显示评论列表
5. 显示相关帖子推荐

### 3.4 评论回复

#### 3.4.1 发表评论

1. 在帖子详情页底部找到评论框
2. 输入评论内容
3. 点击【发表评论】按钮

#### 3.4.2 回复评论

1. 点击评论下方的【回复】按钮
2. 输入回复内容
3. 支持使用@提及其他用户

#### 3.4.3 评论互动

- **点赞**：点击评论右侧的👍图标
- **点踩**：点击评论右侧的👎图标
- **举报**：点击评论右侧的【举报】按钮

### 3.5 帖子互动

#### 3.5.1 点赞/取消点赞

1. 在帖子详情页找到点赞按钮
2. 点击即可点赞或取消点赞
3. 点赞数实时更新

#### 3.5.2 收藏帖子

1. 点击收藏按钮（⭐图标）
2. 收藏成功后按钮变为实心
3. 可在个人中心查看收藏列表

#### 3.5.3 分享帖子

1. 点击分享按钮
2. 选择分享方式：
   - 复制链接
   - 分享到微信
   - 分享到QQ

### 3.6 举报违规内容

1. 点击帖子或评论的【举报】按钮
2. 选择举报原因：
   - 垃圾广告
   - 色情低俗
   - 政治敏感
   - 人身攻击
   - 侵权内容
   - 其他
3. 填写详细描述（可选）
4. 上传证据截图（可选）
5. 提交举报

### 3.7 个人中心

#### 3.7.1 访问个人中心

1. 点击右上角头像
2. 选择【个人主页】或【个人设置】

#### 3.7.2 个人主页

显示以下信息：
- 基本信息：头像、昵称、签名
- 统计数据：发帖数、评论数、获赞数
- 帖子列表：我发布的帖子
- 评论列表：我发表的评论
- 收藏列表：我收藏的帖子

#### 3.7.3 个人设置

**修改基本信息**：
1. 点击【个人设置】
2. 修改昵称、签名、学校等
3. 点击【保存】

**修改头像**：
1. 点击当前头像
2. 上传新头像图片
3. 裁剪调整
4. 点击【确认】

**修改密码**：
1. 进入【账号安全】
2. 输入原密码
3. 输入新密码
4. 确认新密码
5. 点击【确认修改】

### 3.8 消息通知

#### 3.8.1 查看通知

1. 点击顶部铃铛图标🔔
2. 显示未读消息数量
3. 点击进入消息列表

#### 3.8.2 通知类型

| 类型 | 说明 |
|------|------|
| 系统通知 | 系统公告、升级通知等 |
| 评论提醒 | 有人评论了你的帖子 |
| 回复提醒 | 有人回复了你的评论 |
| 点赞提醒 | 有人点赞了你的帖子/评论 |
| @提醒 | 有人@提及了你 |
| 审核通知 | 帖子审核结果通知 |

#### 3.8.3 消息操作

- **全部已读**：标记所有消息为已读
- **删除消息**：点击消息右侧删除按钮
- **清空消息**：清空所有消息

### 3.9 关注用户

#### 3.9.1 关注操作

1. 进入他人个人主页
2. 点击【关注】按钮
3. 成为该用户的粉丝

#### 3.9.2 查看关注列表

1. 进入个人中心
2. 点击【关注】标签
3. 显示所有关注的用户

### 3.10 用户等级

#### 等级说明

| 等级 | 经验值范围 | 特权 |

## 四、版主功能

### 4.1 版主权限说明

版主拥有所辖版块的管理权限，包括：
- 帖子审核
- 帖子置顶/加精/移动/删除
- 版块公告管理
- 用户禁言（限本版块）

### 4.2 帖子审核

1. 进入管理后台 → 待审核帖子
2. 查看待审核帖子列表
3. 点击帖子查看详情
4. 选择操作：
   - **通过**：帖子正常发布
   - **驳回**：填写驳回理由

### 4.3 帖子管理

#### 4.3.1 置顶帖子

1. 在帖子列表或详情页点击【置顶】
2. 选择置顶时长：
   - 1天
   - 3天
   - 7天
   - 永久
3. 确认操作

#### 4.3.2 加精帖子

1. 点击【加精】按钮
2. 帖子会显示精华标识
3. 用户可在精华区查看

#### 4.3.3 移动帖子

1. 点击【移动】按钮
2. 选择目标版块
3. 确认移动

#### 4.3.4 删除帖子

1. 点击【删除】按钮
2. 选择删除原因（可选）
3. 确认删除

### 4.4 举报处理

1. 进入管理后台 → 举报管理
2. 查看举报列表
3. 选择处理方式：
   - **忽略**：无违规，关闭举报
   - **删除内容**：违规，删除帖子/评论
   - **禁言用户**：严重违规，禁言用户

### 4.5 版块公告

1. 进入版块管理
2. 点击【发布公告】
3. 输入公告内容
4. 选择显示时长
5. 发布公告

## 六、常见问题

### Q1: 忘记密码怎么办？

**解决方案**：
1. 在登录页点击【忘记密码】
2. 通过注册邮箱找回密码

### Q2: 为什么我的帖子被删除了？

**可能原因**：
1. 帖子内容违规
2. 被多人举报
3. 版主或管理员删除
4. 系统自动过滤

**解决方案**：
查看消息通知，系统会发送删除通知

### Q3: 如何成为版主？

**条件**：
1. 注册时间超过30天
2. 发帖数量超过20篇
3. 无违规记录
4. 活跃度较高

**申请流程**：
联系管理员申请

### Q4: 为什么无法发帖？

**可能原因**：
1. 版块设置了发帖权限
2. 账号被禁言
3. 内容包含敏感词
4. 版块需要审核

### Q5: 如何修改已发布的帖子？

**操作步骤**：
1. 进入帖子详情页
2. 点击【编辑】按钮
3. 修改内容
4. 点击【保存】

**注意**：部分版块可能限制编辑次数
